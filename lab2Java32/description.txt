Second Java lab, task №32

Task: Упорядочить строки матрицы по неубыванию их наибольших элементов.

Description: для каждой строки найдем ее максимум и сохраним его и номер строки, в которой он находится, 
             в pair (реализован самостоятельно). Отсортируем массив пар при помощи компоратора по первому элементу пары. 
             После выведем их в новом порядке, получая матрицу описанную в условии.
