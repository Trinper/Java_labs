Second Java lab, task №4

Task: Среди строк заданной матрицы, содержащих только нечетные элементы, найти строку с максимальной суммой модулей элементов.

Description: Проходим по каждой строке, подсчитывая количество нечетных элементов строки и ее сумму. 
             Если количество нечетных элементов равняется числу элементов в строке, то сравниваю сумму с 
             максимумом и запоминаю индекс большей. В результате вывожу строку с максимальной суммой.
